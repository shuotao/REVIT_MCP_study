依據 `domain/floor-area-review.md` 執行容積檢討。

## Steps

1. 讀取 `domain/floor-area-review.md` 取得面積分類規則與法規標準
2. `get_active_view` — 確認目前環境
3. `get_all_levels` — 取得所有樓層
4. `get_rooms_by_level` — 逐層取得房間清單與面積
5. 分類房間：
   - 主要空間 → 100% 計入
   - 陽台 → 50% 計入
   - 附屬空間（機房、梯間、電梯間）→ 0%
6. 計算：
   - 各層樓地板面積 = Σ(房間面積 × 計入比例)
   - 總樓地板面積 = Σ(各層)
   - 容積率 = 總樓地板面積 / 基地面積 × 100%
7. 比對法規上限（住宅 300%、商業 600%、工業 200%）
8. 輸出容積檢討報告（含各層明細表）

## Error Handling

| 情況 | 處理 |
|------|------|
| 基地面積未知 | 詢問使用者輸入基地面積（m²） |
| 房間未命名 | 列出未命名房間，請使用者分類 |
| 用途不明 | 詢問使用者確認用途區分（住宅/商業/工業） |
