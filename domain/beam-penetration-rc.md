# RC 混凝土梁穿孔規則

## 1. 幾何限制與計算基準
*   **形狀限制**：僅允許**圓孔**。若為矩形孔，狀態一律判定為 `FAIL`。
*   **長度誤差**：遵循 `beam-penetration-base` 之 $\pm 10 \text{ mm}$ 容許誤差規範（如套管長度與梁寬之比對）。
*   **水平定位起點補償**：
    *   **計算基準**：尺寸標註與法規檢核數值必須為**「套管外孔邊緣」**到**「柱邊或梁端之物理邊緣」**的淨距離 $d$，**嚴禁**以套管中心線計算。
    *   **牆/柱寬度補償**：由於 Revit 預設對齊線位於牆/柱中心，標註與計算必須依據相連牆/柱之寬度，向梁內側進行 `寬度/2` 之物理邊緣補償。

## 2. 水平淨距離檢核 (Horizontal Clearance)
以「梁端（柱邊）之物理邊緣」為起點 ($d=0$)：

### 2.1 大梁檢核規則 (Major Beam)
*   **禁開區 ($d < 1.0H_{major}$)**：距梁端（柱邊）不足 $1.0H_{major}$ 區域禁止開孔。
*   **限制區 ($1.0H_{major} \le d < 1.5H_{major}$)**：最大允許孔直徑 $D \le H_{major}/4$。
*   **一般區 ($d \ge 1.5H_{major}$)**：最大允許孔直徑 $D \le H_{major}/3$。

### 2.2 小梁檢核規則 (Minor Beam)
*   **禁開區 ($d < 0.5H_{minor}$)**：距梁端不足 $0.5H_{minor}$ 區域禁止開孔。
*   **一般區 ($d \ge 0.5H_{minor}$)**：最大允許孔直徑 $D \le H_{minor}/3$。

### 2.3 大小梁接頭避讓規則 (Beam Joint Restriction)
*   **雙重指標並存**：當大梁上的套管靠近與其相交的小梁時，除了需檢核大梁距柱邊的距離之外，亦須檢核套管到該相交小梁物理邊緣的水平距離。
*   **小梁端點避讓限制**：套管至相交小梁物理邊緣的水平淨距離必須 $\ge 0.5H_{minor}$（其中 $H_{minor}$ 為該相交小梁之梁深）。

## 3. 垂直淨距離檢核 (Vertical Clearance)
*   **檢核定義**：套管的**上邊緣到梁頂**之淨距離 $d_{top}$，以及**下邊緣到梁底**之淨距離 $d_{bottom}$，必須符合以下限制：
  * $d_{top} \ge \max(H/3, H_{limit})$
  * $d_{bottom} \ge \max(H/3, H_{limit})$
  * 其中，$H_{limit}$ 對於大梁為 $200\text{ mm}$，小梁為 $150\text{ mm}$。
*   **梁頂增築（增打）厚度排除**：
    *   **判定**：若偵測到梁頂有增築（增打）之厚度，在計算實質梁深 $H$ 及梁頂高時，必須將增築厚度予以扣除（即以實際結構梁深計算）。
    *   **目的**：避免因增打厚度被算入梁深，導致頂底高度檢核超標。

## 4. 相鄰套管淨距離檢核 (Sleeve Spacing Clearance)
*   **檢核定義**：同一支梁上相鄰兩套管之**外邊緣到外邊緣的淨距離** $S_{net}$，必須大於或等於兩孔徑之和：
  * $S_{net} \ge (D_1 + D_2)$
  * **計算式**：$S_{net} = S_{center} - D_1/2 - D_2/2$（其中 $S_{center}$ 為兩套管中心投影之距離， $D_1$ 與 $D_2$ 分別為兩套管之直徑）。
