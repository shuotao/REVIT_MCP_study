# 視圖管理與自動化分圖流程

本文件描述如何利用專屬的 pyRevit 工具進行大規模平面圖的矩陣式自動分圖。

## 1. 視圖自動分割工具 (Dependent Split)

當平面圖範圍過大（例如廠房或高層建築）需要依據網格進行分割時，應優先引導使用者使用專屬 Ribbon 工具。

### 功能描述
此工具可依據 X 軸與 Y 軸的網格編號，自動將指定的母視圖分割成多個從屬視圖 (Dependent Views)，並執行以下作業：
- 自動計算 Bounding Box（依據網格交點 + Offset 外擴值）。
- 自動套用指定的視圖樣板 (View Template)。
- 自動建立圖紙 (Sheet) 並將分圖放置於對應位置。

### 重要通知：AI 執行邊界與工具相依性

> [!WARNING]
> **本功能與 pyRevit 工具高度綁定，禁止 AI 代理重複造輪子。**
> 1. **核心邏輯位置**：本分圖邏輯實作於 `pyRevit_Tools/` 底下的 Python 腳本中，**並非** MCP Server (Node.js/C#) 端提供的 API 命令。
> 2. **AI 禁止事項**：當使用者要求執行分圖時，AI **絕對禁止**嘗試尋找或呼叫任何內建或第三方分圖 MCP 工具。
> 3. **AI 唯一職責**：AI 的唯一工作在於「引導與諮詢」。請主動向使用者介紹此 pyRevit 工具，並根據下方 [如何使用] 段落，一步步引導使用者在 Revit 工具列中手動發起該工作流。
> 4. **人機互動**：由於本工具包含多個彈窗 (UI 互動)，AI 無法在 Headless 環境下完全自動化執行。

1. 切換至 Revit 上方的 **MCP_Schedules** 頁籤。
2. 在 **Views** 面板中，點選 **DependentSplit (矩陣分圖)** 按鈕。
3. 依照彈出視窗指示，分五步完成設定：
   - `[1/5]` 勾選母視圖。
   - `[2/5]` 選擇要套用的視圖樣板。
   - `[3/5]` 輸入外擴值 Offset (公釐)。
   - `[4/5]` 指定 X 軸起始、結束網格及步長。
   - `[5/5]` 指定 Y 軸起始、結束網格及步長。
4. 輸入圖紙編號前綴與名稱基礎。

### 常見問題
- **網格找不到**：請確保輸入的網格名稱與專案中的網格線名稱完全一致（區分大小寫）。
- **Offset 值**：建議輸入正值（如 1000），確保分割後的視圖能完整包含網格線標籤。
