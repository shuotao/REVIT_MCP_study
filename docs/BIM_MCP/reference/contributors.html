CLOSING NOTE

承認函，不是邀請函

為什麼是這種形式
這份名單不是邀請大家來貢獻的公關頁，是一份承認函——這些人各自 fork 出去，按自己的步調寫了東西，後來被逐一手動核對、追依賴閉包、跑跨版本 build、過 QA/QC 才收進 main。名單上每一行都能追到一個 PR 編號、一個 commit，或一個具名的收編波次。他們選擇了 fork 這條路，我們用這頁記下他們做過的事，謝謝。
完整的 fork 盤點方法與資料來源見 docs/fork-audit/（gitignored，內部工作檔）；本頁只呈現對外可公開的致謝內容。

想成為下一位被記在這裡的人？

看看 Contributor Template——先寫 Domain SOP，再視情況寫 Skill 編排，最後才考慮新增工具。三選一決策樹 + 撰寫骨架 + 6 步驟範例都在裡面。
